# 🌍 Overview

FactoryOS is an **AI-powered Industrial Safety Intelligence Platform** designed to monitor factory operations in real time, detect operational risks, reconstruct incidents using an Industrial Time Machine, and generate intelligent recommendations powered by **Google Gemini AI**.

The platform combines telemetry, AI, and visualization into one centralized industrial command center.

---

# 🎯 Problem Statement

Modern factories generate massive amounts of telemetry data every second.

Traditional monitoring systems are:

- Reactive instead of proactive
- Difficult to analyze during emergencies
- Unable to provide intelligent recommendations
- Limited in historical incident investigation

FactoryOS solves these challenges using AI-powered industrial intelligence.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/FactoryOS.git

cd FactoryOS
```

---

## Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```
